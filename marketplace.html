💸 Quando compras 💎 Cristais
O MATIC vai 100% para o cofre do contrato. Os 💎 são creditados no jogo pelo servidor após verificação on-chain. O cofre é a reserva que garante que todos os jogadores podem sempre resgatar os seus 💎.

🏪 Taxas do Marketplace
Todas as taxas das transações dentro do marketplace são divididas: 80% vão para a Pool P2E (que paga recompensas aos jogadores) e 20% vão para manutenção (servidor, Firebase, desenvolvimento).
